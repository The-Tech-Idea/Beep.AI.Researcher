Uploading Documents

Upload PDF, Word, text, and other document files

Supported Formats

You can upload: PDF, Word (.docx), Text (.txt), Markdown (.md), HTML, CSV, and Excel files. Maximum file size is 50MB per file.

How to Upload

  1. Go to Documents in the project sidebar
  2. Drag and drop files onto the upload area, or click to browse
  3. Files will be processed and indexed automatically
  4. Wait for the status to change from Processing to Indexed
