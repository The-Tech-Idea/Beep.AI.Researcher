First Login

Create your account and log in for the first time

Creating Your Account

When you first open the application, you will be directed to the login page. If no admin account exists yet, you can create one during registration.

  1. Click Register on the login page
  2. Enter your username, email, and password
  3. Check your email for a verification link (if email verification is enabled)
  4. Click the verification link to activate your account
  5. Log in with your credentials
Important: The first user registered is automatically granted admin privileges. Admin users can manage other users, configure feature flags, and manage system settings.

Setting Up Multi-Factor Authentication

For enhanced security, you can enable MFA:

  1. Go to Profile
  2. Click Set up MFA
  3. Scan the QR code with your authenticator app (TOTP)
  4. Enter the 6-digit code to confirm
  5. Save your backup codes in a secure location

Your Dashboard

After logging in, you will see the main dashboard with:

  • Your project list with status badges
  • Quick action buttons for creating projects
  • Feature tool grid showing available AI capabilities
  • Recent activity feed

Click New Project to create your first research project.
