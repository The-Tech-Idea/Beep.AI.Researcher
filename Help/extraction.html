Extraction Schemas

Define structured data extraction templates

Creating Schemas

  1. Go to Extraction in the project sidebar
  2. Click New Schema
  3. Enter a schema name and description
  4. Add fields with types: string, number, date, list, or object
  5. Optionally configure validators for each field

Field Types

TypeDescription
StringFree text
NumberNumeric value
DateDate value
ListMultiple values
ObjectNested key-value pairs
