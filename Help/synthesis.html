Running Synthesis

  1. Go to Evidence Synthesis in the project sidebar
  2. Enter your research question
  3. Set the maximum number of evidence items
  4. Click Run Synthesis
  5. Review the AI-generated answer with evidence table

Evidence Classification

Each evidence item is classified as supporting, contradicting, or mentioning the research question.
