Code Browser

Browse and manage hierarchical code trees

Code Tree

The code browser displays codes in a hierarchical tree structure. You can:

  • Create top-level codes with a name, description, and color
  • Create child codes by selecting a parent code
  • Edit or delete existing codes

Code Statistics

Each code shows the number of text references (excerpts) coded with it. Click a code to see all its references.
