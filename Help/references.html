Reference Library

Manage project references with collections and tags

Adding References

  1. Go to References in the project sidebar
  2. Click New Reference
  3. Enter title, authors, publication, year, DOI, and URL
  4. Add notes and tags for organization

Organizing References

Use collections and tags to organize your reference library. You can also search references by query.
