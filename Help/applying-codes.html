Applying Codes

Apply codes to selected text in documents

How to Apply Codes

  1. Open a document in the Document Viewer
  2. Select the text passage you want to code
  3. In the code panel, select or search for a code
  4. Click Apply Code

Coding Memos

You can attach a memo to each code application to record your reasoning or observations about why you coded that passage.
