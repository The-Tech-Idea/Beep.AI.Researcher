Running Extraction

Extract structured data from documents using schemas

Running Extraction

  1. Select a schema from the extraction page
  2. Choose which documents to extract from
  3. Click Run Extraction
  4. Wait for the AI to process the documents
  5. Review results in the extraction table

Reviewing Results

Results are displayed in a table with one row per document. You can edit individual field values and flag incorrect extractions.
