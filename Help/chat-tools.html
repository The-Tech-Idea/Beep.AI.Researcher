Chat Tools

Search library, summarize sources, insert citations

Available Tools

Within chat, you can use several tools:

  • Search Library -- Search your project documents directly
  • Summarize Source -- Get a summary of a specific document
  • Insert Citation -- Generate formatted citations (APA, MLA, Chicago, BibTeX)

Slash Commands

Use slash commands in chat:

  • /generate-flashcards -- Create flashcards from documents
  • /generate-quiz -- Create a quiz from documents
