Creating Projects

Create and manage research projects

Creating a New Project

  1. From the dashboard, click New Project
  2. Enter a project name and description
  3. Select a tenant (if multi-tenancy is enabled)
  4. Click Create

You will be redirected to the project overview page.

Project Start Wizard

For new projects, the Start Wizard guides you through initial configuration:

  1. Document Library -- Select which document library to connect
  2. Answer Style -- Choose fast, balanced, or deep AI retrieval mode
  3. File Type -- Select the most common document type for optimal reading setup
  4. Save -- Apply settings and open the project

Project Status

Projects have status indicators:

  • Ready -- Documents uploaded and indexed
  • In Progress -- Setup partially complete
  • Empty -- No documents uploaded yet
