Quizzes

AI-generated MCQ quizzes from project documents

Creating Quizzes

  1. Go to Quizzes in the project sidebar
  2. Name your quiz and select documents
  3. Set the number of questions
  4. Click Generate Quiz

Taking Quizzes

Click Take Quiz to answer questions interactively. Results are scored with percentage results.
